# Grouping Task

Группировка строк по совпадающим значениям в одинаковых колонках.

## Сборка

```bash
mvn clean package
```
## Запуск

```bash
java -Xmx1G -jar target/grouping-task-1.0.jar <путь_к_входному_файлу>
```
## Результат
output.txt — группы строк в формате:

Группа 1: 
строка1
строка2 ...

Консоль — количество групп с >1 элементом и время выполнения

Результат теста: 



Групп с >1 элементом: 471

Всего групп: 493912
